# Источники: центр «Мой бизнес» и массовые городские события

Дополнение к ТЗ модуля событий (вставка из чата, 2026-07-17).

## 1. Центр «Мой бизнес» Томской области

### Цель
Собирать деловые мероприятия центра «Мой бизнес» Томской области, которые могут привести в Томск предпринимателей, спикеров, партнёров и участников из районов области и других регионов.

### Источник

```yaml
- id: my_business_tomsk
  name: Центр «Мой бизнес» Томской области
  type: html
  base_url: https://www.mb.tomsk.ru/meropriyatiya/
  category: business
  priority: 1
  enabled: true
  refresh_interval_hours: 12
```

### Собирать
название; дату/время; формат; место; адрес; длительность; регистрацию; ЦА; организатора;
ссылку; масштаб; число участников/лимит.

### Правила оценки
| Признак | Спрос |
|---|---|
| Семинар 1–2 ч местный | низкое |
| Очная программа 2+ дня | среднее |
| Межмуниципальный форум | высокое |
| Форум с выставкой | высокое |
| Всероссийский / межрегиональный | критичное |
| Онлайн | не учитывать |

Вне Томска — не в прогнозе автоматически (исключение: проживание в Томске + ручное подтверждение).

## 2. Массовые городские мероприятия

Источники: спорткалендарь, событийный календарь области, библиотеки, РИА (verification_only).

Категории: city_holiday, festival, fair, sports, food, national_culture, family,
exhibition, business, conference, public_holiday.

Не учитывать автоматически: дворовые/школьные/библиотечные лекции, без даты/Томска,
онлайн, аудитория < 100.

## 3. Доп. поля
`is_online`, `registration_required`, `expected_attendance`, `attendance_source`,
`tourism_relevance`, `overnight_likelihood`, `is_public_holiday`.

## 4. Оценка влияния
`event_demand_score = impact_score × overnight_likelihood × duration_factor × audience_scope_factor`

## 5. Уведомления Max
approved AND impact ≥ 80 AND start ≤ сегодня+14д AND overnight ≥ 0.35

## 6. Приёмка
Отдельные сборщики; многодневные = один период; онлайн вне прогноза; вне Томска — вручную;
источник в админке; праздники без иногородней аудитории — не высокий impact.
