# 16.07.2026 — цены конкурентов по объектам

## Проблема
Петровские в вебе: **12 000 ₽** вместо реальных **~2 600 ₽**.

## Причина
Парсер брал только «1 сутки - 12 000» (загородный лот), игнорируя диапазоны вида `2590-3490`.

## Исправление
- Парсинг диапазонов + карточек `.js-product`
- «Цена от» = минимум по объектам (**2590**)
- В БД — строки по объектам (`category`)
- `competitor_category_map` в settings (Петровские → `1room23`)
- В `/competitors` — таблица объектов и Δ к нашей категории
