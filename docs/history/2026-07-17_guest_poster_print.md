# Гостевая афиша недели (печать A4)

Дата: 2026-07-17

## Сделано

- `/events/print?days=10` — лист A4 без навбара/авторизации, `@media print`
- `/events/public?days=10` — мобильная афиша для QR
- Отбор: `approved`, 7–10 дней, ≤10 карточек, гостевые категории,
  дата + площадка + публичная ссылка; без impact/прогноза
- Кнопки в админке: «Печать афиши», «Публичная афиша»
- Конфиг `events.guest_poster` (адрес, Wi‑Fi, контакты, brand)
- QR через `qrcode` (SVG data-URI)
- Модуль `src/events/poster.py`, шаблоны `events_print.html`, `events_public.html`

## Процесс

Понедельник: подтвердить события → открыть «Печать афиши» → Печать / сохранить PDF.
