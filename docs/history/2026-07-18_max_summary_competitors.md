# Рассылка Max: категории и конкуренты

Дата: 2026-07-18

## Что сделано

В ежедневной сводке Max:

1. **Категории** — только занятость по короткому имени (`• 1-КК: N`), без «брон/св» и без «за … ₽».
2. **Итого** — `Итого занято: N (всего M)` + выручка и динамика.
3. **Конкуренты** — новый блок с ценами на сегодня по категориям (или «от» при агрегированной цене).

## Файлы

- `src/notifiers/max_bot.py` — формат секций, сбор конкурентов
- `tests/test_max_bot.py` — ожидания под 3 части сообщения
