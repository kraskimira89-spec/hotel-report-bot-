# Статус «Кандидат» / UX афиши

Дата: 2026-07-17

## Смысл

`candidate` = событие собрано с афиши, **ещё не подтверждено** менеджером.
Не попадает в прогноз и в `/events/print` / `/events/public`.

После ✅ → `approved` («Подтверждено»).

## Исправления UX

- Подпись статуса: «На проверке» вместо «Кандидат»
- Пояснение на странице `/events`
- Кнопки ✅/❌ прямо в списке
- Impact level по-русски (низкое/среднее/…)
- Подсказка на пустой печатной афише
