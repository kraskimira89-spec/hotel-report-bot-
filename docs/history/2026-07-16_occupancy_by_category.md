# 16.07.2026 — статус номеров по категориям

## Запрос
Вместо «Все категории: 19/25/0» — разбивка св/зан/брон по типам.

## Решение
- WebPMS `/rooms` → фонд по `roomTypeId`
- analytics `reservationId` = `roomStay.id` → занятость
- `CheckedIn` → зан, `New` → брон
- карта `travelline.room_type_id_map` в `settings.yaml`

## Пример (16.07.2026)
Итого 20 / 21 / 3 (всего 44), загрузка 56.82% (25/44).
