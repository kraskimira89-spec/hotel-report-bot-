# Промпты для ИИ-аналитики

Каждый файл — отдельный системный/задачный промпт для YandexGPT. Правьте текст здесь — он подхватывается аналитикой.

- `00_system_base.md` — база: роль, правила достоверности, приоритет источников (TravelLine), правовая модель, формат. Подставляется как system-часть ко всем задачам.
- `01_numeric_analytics.md` — аналитика цифр (загрузка, ADR, RevPAR, каналы, конкуренты).
- `02_reviews_classification.md` — классификация отзывов на 3 категории + сводка.
- `03_recommendations.md` — рекомендации к действию на основе данных.

Принцип: `00_system_base` + один задачный промпт = полный контекст для конкретной задачи.

## Общие правила оформления

- **Формат дат** — всегда ДД.ММ.ГГГГ (например, 17.07.2026). Период — через тире: 10.07.2026–16.07.2026. Сравнение периодов — слово «против»: «10.07.2026–16.07.2026 против 03.07.2026–09.07.2026». Не использовать форматы ГГГГ-ММ-ДД или ДД/ММ в тексте для владельца.
